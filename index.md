## What is HDCRS V2?

A **zero‑cost, open‑source** web platform that automates the entire daily supervision workflow for Halbeeg’s construction sites.  
It enforces **every rule** of the Supervision Operational Manual v2.0 – from contract verification to final archiving – while freeing the Project Manager from hours of manual report compilation.

### Key Highlights

- 📱 **Simple phone‑friendly form** for Resident Engineers – no more WhatsApp reports.
- ✅ **Mandatory Director approval** before any client report is sent (Gate G7).
- 🔍 **Hold‑point inspections** tracked with electronic signatures.
- 💰 **Two‑step IPC verification** prevents over‑certification.
- 📊 **Live KPI dashboards** feed directly into performance evaluations.
- 🗄️ **Automatic archiving** – nothing stored on personal phones.
- 💵 **Total cost: $0** – built entirely on free Google tools and GitHub Pages.
